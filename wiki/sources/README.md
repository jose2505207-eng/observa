# Sources

Research and reference material that informs OBSERVA's design.

## Layout
- `raw/` — unmodified captures: PDFs, datasheets, saved articles, benchmark dumps, transcripts. Keep the original so claims can be re-verified.
- `processed/` — distilled notes following [[source-template]], one file per source.

## Rules
- Every claim in an architecture or model page that depends on external facts (ExecuTorch behavior, QNN support, model benchmarks) should trace back to a processed source here, or be tagged **Research** until it does.
- Don't paraphrase a source into a design page without leaving a note here.
