# Demo Checklist

> Run through this before presenting. Keep it boring and reliable.

## Device prep
- [ ] Target device charged; spare/charger on hand.
- [ ] App installed via `./gradlew installDebug` and launched once ([[build-and-run]]).
- [ ] Camera permission already granted (don't fumble the prompt live).
- [ ] Volume up; TTS/audio path verified (once it exists).
- [ ] Screen timeout extended so the device doesn't sleep mid-demo.

## Scene prep
- [ ] Known props ready (a sign/label for OCR, an obstacle/person for awareness).
- [ ] Lighting adequate for the camera.

## The offline proof
- [ ] Airplane mode toggled **on stage** so judges see it ([[airplane-mode-demo]]).
- [ ] Frame counter / awareness output visibly live after airplane mode.

## Story
- [ ] [[judge-script]] rehearsed; honest status line ready.
- [ ] [[failure-recovery]] reviewed.

## Backup
- [ ] Pre-recorded video of the working flow as fallback.
